# Application Icon Placeholder

Place your application icon here as `icon.ico`.

## Creating an Icon

You can create an icon using:
- Online tools like https://www.favicon-generator.org/
- Desktop tools like GIMP, Photoshop, or IcoFX
- Convert from PNG: Use ImageMagick or online converters

## Icon Requirements

- Format: .ico
- Recommended sizes: 16x16, 32x32, 48x48, 256x256
- File name: icon.ico

## Default Icon

If no icon is provided, the application will use the default Java/JavaFX icon.
